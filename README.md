# Johnson Pig Farm

Premium Pork & Sustainable Waste Solutions | Kerala

A modern, Awwwards-winning cinematic React and Framer Motion website built with an ultra-premium, minimalist design aesthetic.

## Features
- **React + Vite Architecture**: Lightning-fast local development and production builds.
- **Framer Motion Animations**: Hardware-accelerated, highly interactive scroll and hover animations.
- **Cinematic Canvas Scrubber**: A 90-frame drone sequence perfectly synced with scroll progression.
- **Lenis Smooth Scroll**: Butter-smooth scrolling interpolation.
- **Tailor-Made Light Theme**: Deep frosted glassmorphism UI elements over a clean `f5f5f7` canvas.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
